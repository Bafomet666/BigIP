# BigIP

Установка ненужна, заходите и запускаете.

Заходите в папку с инструментом, далее :
./satana_locator_0.5.pl

Использовать:

./satana_locator_0.5.pl google.com

./satana_locator_0.5.pl 216.58.209.150
